DNS

What does DNS mean?

DNS stands for "Domain Name System". This is a hierarchical naming system for computers.


How do we use it?

DNS associates names with its corresponding address. For example when you type on the search bar "facebook.com", your computer is using the Domain Name System to look up domain names and get the associated IP address. You can picture DNS as a phone directory; you type the name of the person you want to reach, and your phone will look for the specific phone number that will get you on a call with the person you want to talk to.

We use multiple DNS servers that are connected in a distributed hierarchy. This makes it easier for the DNS to lead you to the website you want to visit. For example, we have DNS servers that will handle .com, while we have other that will handle .org or .net.

Let's say you want to go to amazon.com. DNS will find the corresponding IP address to your "amazon.com" request. Once DNS has given you the IP address for the website you want to reach, your web server will use this IP address to lead you to Amazon's website!
